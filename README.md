# ev3_arm
Robot control demo with EV3

Requirements:

nengo 

 - `pip install nengo`

nengo_gui (on `hooks` branch)

 - `git clone https://github.com/nengo/nengo-gui`
 - `cd nengo-gui`
 - `git checkout hooks`
 - `pip install -e .`

nengo_learning_display

 - `git clone https://github.com/tcstewar/nengo_learning_display`
 - `cd nengo_learning_display`
 - `pip install -e .`
